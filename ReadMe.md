# Task manager

REST API that allows user to manage tasks. The code was written using TDD.

## Clone the project

```
$ git clone https://github.com/JacekKaczmarek10/Task-manager.git
$ cd Task-manager
```

## Run project

```
$ cd Task-manager
$ go build
$ go run .
```


## Run test

```
$ cd Task-manager
$ ./test.sh
```
